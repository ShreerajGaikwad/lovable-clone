💖 Lovable Clone

A modern web application inspired by Lovable, built to demonstrate full-stack development skills including authentication, user interactions, and responsive UI.
This project showcases real-world application architecture, clean code practices, and scalable design.

📱 Responsive design for mobile and desktop
